# 전자정부 표준프레임워크 4.3.0 컨트리뷰션

셀레늄 화면 자동화 테스트

셀레늄은 웹 애플리케이션 자동화 및 테스트를 위한 포터블 프레임워크이다. 셀레늄은 테스트 스크립트 언어를 학습할 필요 없이 기능 테스트를 만들기 위한 플레이백 도구를 제공한다.
